### 5. API Integration & Data Fetching

**Axios instance**
- axios.js
- Base URL: `http://localhost:5000`
- Sets `Content-Type: application/json`
- Adds `Authorization: Bearer <token>` header when `token` exists in `localStorage`

**Network endpoints**
- Auth
  - `POST /auth/login`
  - `POST /auth/register`
- Stores
  - `GET /stores`
- Ratings
  - `POST /ratings`
  - `PUT /ratings/store/:ratingId`

**Data fetching pattern**
- Redux Toolkit async thunks perform API calls
- Thunks return `response.data` or reject payloads with error messages
- UI components dispatch thunks and read Redux state

**Loading / error handling**
- Login.jsx
  - disables submit button when `loading`
  - displays `error` from auth slice
- Register.jsx
  - disables submit button when `loading`
  - displays `error`
- StoreList.jsx
  - shows loading text when `stores.loading`
  - shows error text when stores fetch fails
  - no centralized error boundary or global notification system

**API error handling**
- Thunks extract:
  - `error.response?.data?.message`
  - fallback message string
- Errors are stored in slice state and displayed in pages
